--- Utilizando Prompts para Gerar Insights de Relatórios de Vendas

-- Introdução
Este projeto faz parte do desafio da DIO e tem como objetivo aplicar engenharia de prompts em ferramentas de inteligência artificial para analisar relatórios de vendas, identificar 
padrões e gerar insights estratégicos para o negócio.  

--- Objetivos
- Explorar o uso de prompts para interpretar dados de vendas.  
- Consolidar e organizar datasets em formato legível.  
- Gerar insights relevantes para apoiar a tomada de decisão.

--- Processo do Projeto
- Base de Dados
- Criada em data/base_de_dados.md.  

--- Processamento 
- Script scripts/process_d descreve como consolidar os dados.  

--- Aplicação de Prompts
- Exemplos de prompts em insights/insights.md.  
- Insights documentados.  

--- Relatórios Self-Service 
- Descrição em reports/relatorio_self_service.md.  
- Versão paga: Power BI Pro.  
- Versão gratuita: Google Data Studio ou similares.  

--- Exemplos de Insights Obtidos
- Top produtos mais vendidos: Jogo A, Jogo B, Jogo C.  
- Região com maior faturamento: Sudeste.  
- Período de maior demanda: Novembro e Dezembro.  
- Recomendação estratégica: aumentar estoque e campanhas promocionais em novembro/dezembro.
